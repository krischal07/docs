## Recommended Flow

<Steps>
  <Step title="Read Overview">
    Review the native overview and architecture before wiring any endpoints.
  </Step>
  <Step title="Confirm Auth">
    Use the native authentication guide to confirm partner headers, integration-key usage, and webhook auth.
  </Step>
  <Step title="Connect Merchant">
    Use the Connection Key and Partner API guides to connect the merchant.
  </Step>
  <Step title="Sync Locations">
    Sync external locations and resolve any review-required mapping issues.
  </Step>
  <Step title="Verify Customer Flow">
    Use Customer Identity and Customer API to validate search and retrieval behavior.
  </Step>
  <Step title="Send Test Sale">
    Submit the initial sale through the native partner route and confirm the event is accepted.
  </Step>
  <Step title="Complete Readiness Checks">
    Use the readiness checklist and testing guide before switching to production traffic.
  </Step>
</Steps>
