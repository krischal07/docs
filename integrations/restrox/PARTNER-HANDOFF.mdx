## Supported Events

<Columns cols={3}>
  <Card title="order.completed" icon="shopping-bag">
    Send completed sales for loyalty evaluation.
  </Card>
  <Card title="refund.created" icon="rotate-ccw">
    Reverse loyalty activity tied to an earlier sale.
  </Card>
  <Card title="order.voided" icon="x-circle">
    Notify Samparka when a previously sent sale has been voided.
  </Card>
</Columns>

## Recommended Flow

<Steps>
  <Step title="Read Overview">
    Review the onboarding map and the canonical guide before configuring anything.
  </Step>
  <Step title="Configure Webhook">
    Use the webhook endpoint and authentication guides to set up delivery.
  </Step>
  <Step title="Send Test Sale">
    Follow the quick start and testing guide to send the initial `order.completed` payload.
  </Step>
  <Step title="Test Duplicate Handling">
    Repost the same payload once and confirm the duplicate-safe response.
  </Step>
  <Step title="Test Refund">
    Send the refund payload that reuses the original sale identifier.
  </Step>
  <Step title="Complete Checklist">
    Run the production-readiness checks before switching to live traffic.
  </Step>
  <Step title="Go Live">
    Use the checklist and verification materials as your signoff package.
  </Step>
</Steps>
