Welcome to the RestroX outlet-owned integration path for Samparka.

<Steps>
  <Step title="Create The Integration">
    Merchant creates the integration for a specific outlet with `provider`, `storeId`, and `outletId`.
  </Step>
  <Step title="Share Credentials">
    Merchant shares the Integration Key with RestroX. Samparka also generates the webhook token at creation time.
  </Step>
  <Step title="Connect The Restaurant">
    RestroX calls `POST /api/partners/restrox/connect` and sends the restaurant identifier that belongs to the outlet.
  </Step>
  <Step title="Validate Customer Flow">
    Use the customer APIs only for read-only customer lookup inside the resolved store.
  </Step>
  <Step title="Send A Valid Sale">
    Send the first valid sale through the webhook token URL. Samparka promotes the integration from `CONNECTED` to `ACTIVE`.
  </Step>
</Steps>
