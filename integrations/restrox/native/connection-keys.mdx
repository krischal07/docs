Samparka generates the RestroX Integration Key when the merchant creates the outlet-owned integration.

## What The Key Does

- resolves the correct `PosIntegration`
- lets RestroX connect the restaurant
- scopes partner test-sale calls
- scopes customer lookup requests

## What The Key Does Not Do

- it does not replace the webhook token
- it does not identify multiple outlets at once
- it does not create a second integration for the same outlet

## Reuse Rules

- reconnect keeps the same Integration Key
- rebind keeps the same Integration Key
- key rotation generates a new Integration Key and clears `integration_key_last_used_at`
