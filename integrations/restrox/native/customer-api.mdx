RestroX customer routes are read-only.

## Supported Routes

- `GET /api/partners/{provider}/customers/search?phone=9801234567`
- `GET /api/partners/{provider}/customers/{customerId}`

## Authentication

- `x-partner-key`
- `x-integration-key`

## Behavior

- provider must be `restrox`
- customer search supports exact phone lookup only
- the Integration Key determines which store is searched
