---
title: Readiness Checklist
description: Confirm an outlet-owned RestroX integration is ready for production traffic.
sidebarTitle: Readiness Checklist
---

- [ ] Store already has the target outlet
- [ ] RestroX integration created with `storeId` and `outletId`
- [ ] Integration Key copied to RestroX
- [ ] Partner connect completed successfully
- [ ] `external_location_id` is stored on the integration
- [ ] Webhook URL uses the integration webhook token
- [ ] First valid sale has been received
- [ ] Integration lifecycle status is `ACTIVE`
- [ ] Health status is `HEALTHY`
