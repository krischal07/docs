title: Error Codes
description: Review the partner-visible RestroX error responses documented in the current implementation.
sidebarTitle: Error Codes
---

| Route Family | Status | Message |
| ------ | ------ | ------- |
| Partner auth | `503` | `RestroX partner API is not configured` |
| Partner connect | `404` | `Invalid Integration Key` |
| Partner connect | `400` | `RestroX restaurant identifier is required to connect` |
| Partner connect / rebind | `409` | `Disconnect the integration before rebinding it to another restaurant` |
| Partner connect duplicate | `409` | `This RestroX restaurant is already bound to another Samparka outlet` |
| Test sale | `409` | `This RestroX integration is not connected to a restaurant yet` |
| Test sale | `409` | `The provided restaurant identifier does not match the connected outlet` |
| Webhook | `404` | `Invalid webhook token` |
| Webhook | `409` | `Integration disconnected` |
| Webhook | `409` | `Integration is not connected to a restaurant` |
| Webhook | `409` | `Restaurant binding mismatch` |
