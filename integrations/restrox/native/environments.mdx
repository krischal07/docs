The docs package uses placeholder hosts unless your team has supplied environment-specific domains.

## Documented URL Shapes

- merchant APIs: `https://your-domain/api/pos-integrations`
- partner APIs: `https://your-domain/api/partners/restrox`
- webhook ingress: `https://your-domain/webhook/restrox/{token}`

## Important Constraint

Environment names are not hard-coded in the RestroX contract. The important requirement is that every environment preserves the same outlet-owned route shapes and field names.
