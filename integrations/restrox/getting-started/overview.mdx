## Recommended Reading Order

<Steps>
  <Step title="Start With Native Overview">
    Read the native overview and architecture so the full system makes sense before you start coding.
  </Step>
  <Step title="Read Connection And Auth">
    Review Connection Keys, authentication, and environments before implementing any client behavior.
  </Step>
  <Step title="Implement Onboarding">
    Use Merchant Onboarding and Store Linking to connect the merchant and sync locations.
  </Step>
  <Step title="Implement Customer Flow">
    Use Customer Identity and Customer API to make customer resolution deterministic.
  </Step>
  <Step title="Validate Events">
    Use the Testing Guide, Readiness Checklist, and webhook transport docs to confirm the integration is ready.
  </Step>
</Steps>
