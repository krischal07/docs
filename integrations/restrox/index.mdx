## Onboarding Flow

<Steps>
  <Step title="Receive Token">
    Confirm the webhook URL and token from Samparka, along with outlet mapping details.
  </Step>
  <Step title="Configure Webhook">
    Point RestroX at the Samparka webhook endpoint and send JSON `POST` requests.
  </Step>
  <Step title="Send Test Sale">
    Use the documented sale fixture to verify that an accepted request returns a `200` acknowledgment.
  </Step>
  <Step title="Verify Response">
    Confirm the expected success message and resend the same payload once to validate idempotency.
  </Step>
  <Step title="Test Refund">
    Send a `refund.created` payload that references the original sale identifier.
  </Step>
  <Step title="Go Live">
    Finish the integration checklist, then promote the connection to production traffic.
  </Step>
</Steps>
