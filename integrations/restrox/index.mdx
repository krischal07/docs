## Native Onboarding Flow

<Steps>
  <Step title="Receive Connection Key">
    The merchant creates a RestroX integration in Samparka and shares the Connection Key with RestroX.
  </Step>
  <Step title="Connect Merchant">
    RestroX calls the native partner connect route using the Connection Key and partner credentials.
  </Step>
  <Step title="Sync Locations">
    RestroX syncs external locations so Samparka can match them to merchant outlets.
  </Step>
  <Step title="Verify Customer Flow">
    RestroX verifies customer lookup and upsert behavior using normalized phone identity.
  </Step>
  <Step title="Send Test Sale">
    RestroX sends a test sale through the native partner API and confirms the webhook-backed event engine accepts it.
  </Step>
  <Step title="Go Live">
    Once the integration is ready, production sales, refunds, and voids continue through the webhook-backed event flow.
  </Step>
</Steps>
