# RestroX Partner Guide

This root-level guide mirrors the canonical outlet-owned RestroX docs in [`integrations/restrox`](/Users/krischalshrestha/Documents/work related/samparka/docs/integrations/restrox/README.md).

RestroX is an outlet-owned integration in Samparka:

- create with `storeId` and `outletId`
- connect one restaurant per outlet
- route webhooks with `POST /webhook/restrox/{token}`
- activate on the first valid sale
- verify customer creation or resolution before signoff
- verify loyalty outcome and awarded points before signoff
