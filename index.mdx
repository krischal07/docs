## How To Use This Portal

<Steps>
  <Step title="Choose An Integration">
    Start from the integration card that matches the platform you are connecting to Samparka.
  </Step>
  <Step title="Follow The Guide">
    Each integration section can include its own getting-started flow, payload requirements, test cases, and reference assets.
  </Step>
  <Step title="Use Canonical Assets">
    Download the integration-specific OpenAPI, Postman, and example payload assets from within that integration section.
  </Step>
  <Step title="Return Here For New Guides">
    As more Samparka integrations are published, they will be added to this homepage.
  </Step>
</Steps>
